## Compliance Framework

### Personal Data Protection Act 2010 (PDPA)
- **Section 130**: Data processor obligations (see DPA)
- **Section 129**: Cross-border transfer safeguards (Stripe to Singapore/US)
- **Section 44**: Statutory disclosure exemption (IRBM e-Invoice submission)
- **Section 5**: User rights (access, correction, withdrawal, complaints)

### Regulatory Authorities
- **PDPA Commissioner**: For data protection complaints
- **IRB Malaysia**: E-Invoice compliance via Peppol Network
